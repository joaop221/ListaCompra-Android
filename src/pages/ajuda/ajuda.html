Ajuda Como adicionar uma lista? Bastar clicar no botão na parte inferior com símbolo de "+". Uma nova janela com a descrição do item será aberta para adicionar um novo item. Como criar um grupo? Bastar clicar no botão na parte inferior com símbolo de "+". Uma nova janela com a descrição do grupo será aberta para adicionar um novo grupo. Como adicionar um participante ao grupo? Bastar clicar no botão na parte superior com símbolo de "+". Uma nova janela com os dados do participante será aberta.
